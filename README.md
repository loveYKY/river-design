### 安装最后已经发布的组件库来试试

```js
npm install river-design --save
```

### 使用文档

```
https://loveyky.github.io/river-design-doc.github.io/?path=/docs/alert--default
```

### 加载样式

```
import 'river-design/dist/index.css' 
```

### 引入组件

```
import { Button } from 'river-design'
```

### 启动本地环境

```
npm run stroybook
```

### 跑单元测试

```
npm test
```

### build可发布静态文件

```
npm run build
```

### 发布到 npm

```
npm run publish
```
